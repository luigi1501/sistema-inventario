# 📦 Sistema de Gestión de Inventario Pro

Este es un sistema de gestión de inventario desarrollado con **Node.js**, **Express** y **TiDB Cloud (MySQL compatible)**. Permite el control de stock, categorías y visualización de ventas en tiempo real.

## 🚀 Características
- **Base de Datos Distribuida:** Conexión segura con TiDB Cloud (AWS Virginia).
- **Gestión de Productos:** CRUD completo (Crear, Leer, Actualizar, Eliminar).
- **Control de Inventario:** Cálculo automático de valor total y alertas de stock.
- **Seguridad:** Uso de variables de entorno y conexión SSL/TLS.

## 🛠️ Tecnologías
- **Backend:** Node.js, Express.
- **Frontend:** EJS, Bootstrap, JavaScript.
- **DB:** TiDB Cloud (MySQL compatible).
